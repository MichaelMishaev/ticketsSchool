## FAQ ANSWERS (Based on Implementation)

**Can we manage multiple events?**
Yes - unlimited events on free plan (until quota reached), each with separate registrations and settings.

**Do attendees need to create accounts?**
No - they can register with just a phone number or email, no login required.

**Can we set registration limits?**
Yes - set total capacity and max spots per person. Automatic waitlist when full.

**Can we customize the registration form?**
Yes - add unlimited custom fields per event (text, dropdown, checkbox, etc).

**Can we see who registered?**
Yes - complete registration list with all data, searchable and exportable to CSV.

**Can team members help manage?**
Yes - invite them with specific roles: ADMIN (full access), MANAGER (edit registrations), VIEWER (read-only).

**Is it secure?**
Yes - encrypted logins, password resets, data isolation by school, team permissions enforced.

**What if the event fills up?**
Registrants automatically join the waitlist. You can promote waitlist members to confirmed spots if capacity opens.

**Can we change event details?**
Yes - update almost everything except the event URL. You can pause or close registration anytime.

**Can we delete an event?**
Yes, but only if no one has registered yet. This protects attendee data.
